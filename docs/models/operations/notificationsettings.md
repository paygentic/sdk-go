# NotificationSettings

Notification preferences for this customer. Only provided fields are updated.


## Fields

| Field                                                     | Type                                                      | Required                                                  | Description                                               |
| --------------------------------------------------------- | --------------------------------------------------------- | --------------------------------------------------------- | --------------------------------------------------------- |
| `InvoiceIssued`                                           | `*bool`                                                   | :heavy_minus_sign:                                        | Whether to send invoice issued emails to this customer.   |
| `InvoicePaid`                                             | `*bool`                                                   | :heavy_minus_sign:                                        | Whether to send invoice paid emails to this customer.     |
| `RenewalReminder`                                         | `*bool`                                                   | :heavy_minus_sign:                                        | Whether to send renewal reminder emails to this customer. |
